Browser Object Model (BOM) - Quick Notes

BOM allows JavaScript to interact with the browser window.

  • location.reload() — Reloads the current page.
  • location.href — Changes the current URL.
  • history.back() or history.go(-n) — Goes back in history.
  • history.forward() or history.go(n) — Goes forward in history.
  • window.innerHeight — Viewport height (excluding toolbars).
  • window.innerWidth — Viewport width.
  • window.open("url", "name", "options") — Opens a new browser window/tab.
  • window.close() — Closes the current window (only if it was opened via script).
  • resizeBy(x, y) — Resizes the window by a relative amount.
  • resizeTo(x, y) — Resizes the window to specific dimensions.
  • moveBy(x, y) — Moves the window by a relative amount.
  • moveTo(x, y) — Moves the window to specific screen coordinates.
  • scrollBy(x, y) — Scrolls the document by given pixels.
  • scrollTo(x, y) — Scrolls to a specific position in the document.
  • print() — Opens the print dialog for the page.
  • document — Entry point to the DOM (Document Object Model).
